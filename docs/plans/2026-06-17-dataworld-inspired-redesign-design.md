# data.world-Inspired Portfolio Redesign Design

## Goal

Apply a site-wide visual refresh inspired by data.world's enterprise data-platform look while keeping the portfolio clearly branded around Kent John U. Macalam and his project proof.

## Reference Notes

The reference direction uses a dark navy hero, bold white headline, warm yellow call-to-action, clean navigation, and product-style sections that frame data discovery, governance, connection, and clarity.

This redesign should use those ideas as inspiration only. It should not copy data.world's brand, logo, layout, exact wording, or proprietary assets.

## Approved Scope

Apply the redesign to the whole portfolio:

- `index.html`
- `style.css`
- `projects/life-of-a-bill.html`
- `projects/from-farm-to-cup.html`
- `projects/smartrip-cdo.html`

## Visual Direction

- Replace the current all-dark portfolio feel with a dark enterprise-blue hero/header and light, clean content bands below.
- Use yellow as the primary CTA accent, supported by blue and teal for data/AI cues.
- Add a slim announcement/data-status bar at the top of the homepage and detail pages.
- Make project cards look more like product feature cards: white surfaces, stronger hierarchy, crisp shadows, and clear hover behavior.
- Preserve all existing project proof content, full-card navigation, case-study pages, resume downloads, GitHub links, and live app links.

## Interaction Direction

- Featured project cards remain fully clickable to detail pages.
- Nested GitHub/live links remain separately clickable.
- Detail pages keep their current case-study structure but inherit the refreshed palette and card style.

## Verification

Verify:

- Style tokens use an enterprise-blue/yellow/light-surface system.
- Homepage and detail pages include the new announcement/status bar.
- Project cards still navigate to their detail pages.
- All three detail pages still load with HTTP 200.
- No horizontal overflow on desktop or mobile.
- Browser console has no errors.
